# AppXplorer
A simple Android app that displays a list of installed applications with detailed information including name, version, package name, APK checksum, and an option to launch the app.

## Built With
1. Kotlin
2. Jetpack Compose
3. Material 3 Design
4. Clean Architecture + MVI
5. Coroutines
6. Flow
7. Hilt
8. Coil
